Mountain Browser

A modern, native web browser for Apple TV — built with SwiftUI and SwiftData.

GitHub

Features

🔍

Integrated Web Search

DuckDuckGo search with web, image, and video results. No API keys required.

📖

Reader Mode

Web pages rendered as native readable text — optimized for TV display on the big screen.

🌍

Wikipedia Integration

Automatic Wikipedia infoboxes with multi-level detail views and multi-language support.

📋

Tab Management

Up to 8 simultaneous tabs in a Safari-style grid layout with real-time status.

🎮

Dual Navigation Modes

Scroll mode (focus-based) and cursor mode (touchpad-based) — the right control for every use case.

🎬

Video Playback

Direct playback of MP4, M3U8, and MOV files. Plus embedded YouTube and Vimeo videos.

🔖

Bookmarks & History

Persistent bookmarks with folder support and automatic browsing history.

⚙

Settings

JavaScript, cookies, pop-up blocker, navigation bar, and Wikipedia language individually configurable.

🌐

5 Languages

Full localization: German, English, French, Spanish, and Italian.

Technology

SwiftUI SwiftData WebKit AVKit Combine tvOS 17.0+ Swift 5.10 Xcode 15+
