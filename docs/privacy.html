Privacy Policy

Stefan Friedrich

2. Overview

Mountain Browser is a web browser for Apple TV (tvOS). Protecting your data is important to us. This privacy policy explains what data is processed when you use the app.

Summary: Mountain Browser does not collect, store, or transmit any personal data. All data remains exclusively on your Apple TV device.

3. No Tracking

Mountain Browser does not include any tracking. No analytics or advertising SDKs are used. No user profiling takes place. The app does not collect device IDs or any other identifiers for tracking purposes.

4. Locally Stored Data

The following data is stored exclusively on your device and is never transmitted to any server:

  • Bookmarks — Saved URLs with folder structure
  • Browsing History — Visited URLs with timestamps
  • Tabs — Open tabs and their state
  • Search Results — Cached search results per tab
  • Settings — Your preferences (JavaScript, cookies, pop-up blocker, etc.)
  • Cookies — Cookies set by websites (can be disabled in settings)

Data is stored via SwiftData (local) and UserDefaults. No cloud synchronization takes place — iCloud/CloudKit is explicitly disabled.

5. Third-Party Services

Mountain Browser uses the following external services for core functionality:

DuckDuckGo

Search queries are sent to DuckDuckGo for web search. According to their privacy policy, DuckDuckGo does not collect personal data or create user profiles. More information: duckduckgo.com/privacy

Wikipedia

Requests are sent to the Wikipedia REST API for knowledge information (infoboxes). The Wikimedia Foundation's privacy policy applies: Wikimedia Privacy Policy

Visited Websites

When visiting websites through the browser, data is exchanged with the respective web server (IP address, HTTP headers, etc.). The privacy policies of the respective website operators apply.

6. No Data Collection

Mountain Browser does not collect any personal data. No data is transmitted to the developer, to Apple, or to any third party — except for the services mentioned in Section 5, which are required for functionality.

7. Data Deletion

You can delete all locally stored data at any time by:

  • Deleting individual bookmarks or the entire history within the app
  • Uninstalling the app from your Apple TV — this completely removes all app data

8. Changes

This privacy policy may be updated from time to time. Changes will be published on this page with the date of the last update.
